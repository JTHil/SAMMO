sammo.extractors#

This module contains components that extract data from the output of other components, typically from LLM calls. Common formats such as JSON, XML, or Markdown are supported and require no data format specification. If validation is required, it should happen downstream of the extraction step.

Module Contents#

Classes#

Extractor

Base class for all extractors. Extractors take the output of a child component and extract data from it.

DefaultExtractor

Performs no processing. Passes value(s) through.

SplitLines

Splits a string into lines.

StripWhitespace

Strips whitespace from a string.

LambdaExtractor

Extracts data from the output of a child component using a lambda expression. This is useful if minor processing is required, e.g. to lowercase the output. The lambda expression need to be a string so it can be serialized. It must take exactly one argument.

ParseJSON

Extracts all matches of a regular expression. Supports object or list fragments surrounded by text.

ExtractRegex

Extracts all matches of a regular expression.

MarkdownParser

Parses text as simplified Markdown only separating sections and paragraphs.

YAMLParser

Parses text as YAML, but does not support fragments due to lack of delimiter in this format.

ParseXML

Parses text as XML, supporting fragments that are surrounded by text.

JSONPath

Extracts all matches of a JSONPath expression.

ToNum

Converts the output of a child component to a number * factor + offset.
